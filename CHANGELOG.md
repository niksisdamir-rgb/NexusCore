## [1.0.0] - 2026-04-13

### Added
- **3D Interactive Plant Visualization**: Real-time rendering of Elkonmix-90 with three.js, including animated silos, mixer, and materials flow.
- **SCADA Dashboard**: Comprehensive control center with 11 functional modules (Production, Recipes, Inventory, Delivery Notes, etc.).
- **NexusCore AI Integration**: Intelligent agent system powered by Gemini for automated recipe extraction and plant optimization.
- **Real-time Telemetry**: SSE-based streaming for zero-latency sensor updates.
- **Inventory Management**: Automated stock depletion logic and low-stock alerting systems.
- **Authentication & Security**: Role-based access control (RBAC) with NextAuth.js and Prisma security.
- **Landing Page**: Professional marketing site with technical specs, FAQ, and 3D preview.

### Changed
- Migrated core logic to Next.js 16 with App Router.
- Refined UI with Tailwind CSS v4 and shadcn/ui.
