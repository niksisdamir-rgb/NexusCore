## Reporting a Vulnerability

We take the security of NexusCore seriously. If you discover a security vulnerability, we would appreciate it if you could let us know immediately. 

Please do **NOT** report security vulnerabilities via public GitHub issues.

Instead, please send an email to security@example.com (replace with valid email) or disclose via GitHub's Security Advisory feature if enabled on the repo. We will acknowledge receipt of your vulnerability report and will strive to send you regular updates about our progress. If you're interested, we'll coordinate public disclosure with you so that you can get credit for the discovery.
